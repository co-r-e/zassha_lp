Five-minute publishing workflow

  1. 01

    Capture

    Record at 1080p+ using macOS or Windows screen capture tools for best accuracy.

  2. 02

    Upload

    Drop the file into ZASSHA and pick Summary or Detail mode depending on the level of guidance you need.

  3. 03

    Hint

    Add a short goal statement so AI can focus on the outcome you expect.

  4. 04

    Review

    Inspect the overview, business inference, and operation table with hoverable thumbnails.

  5. 05

    Export

    Send the manual to DOCX, PPTX, or XLSX with consistent naming that matches your source video.

Questions

Is ZASSHA free to use?

Yes. The open-source app is free—you only need a Gemini API key, and any usage charges come directly from Google.

Which languages can ZASSHA output?

English and Japanese manuals are supported out of the box, and you can switch between them using the header language menu.

How secure is my recording?

Processing runs through Google's Gemini Files API. Video segments are only stored temporarily when server-side segmentation is enabled.

Do I need to edit screenshots manually?

No. ZASSHA captures thumbnails automatically for every detected operation, so you only fine-tune the copy.
